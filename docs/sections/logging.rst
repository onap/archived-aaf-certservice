Logging
=======

Where to Access Information
---------------------------

Certification Service logs are available in the Docker container

    docker exec -it aaf-certservice-api bash

Path to logs:

    /var/log/onap/aaf/certservice

Available log files:

    * audit.log
    * debug.log
    * error.log
